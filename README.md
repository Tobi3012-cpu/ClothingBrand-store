# STREETIFY • SARTREET

![Streetify Banner](https://via.placeholder.com/1200x400/0f0f0f/d71e1e?text=SARTREET+FREE+YOUR+STYLE)

A modern, responsive multi‑page e‑commerce website for a streetwear brand. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools. Features a persistent shopping cart, product modals, and a bold red‑white‑black design.

## 🧩 Features

- **7 fully functional pages** – Home, Shop, Latest Drops, Services, About, Contact, Checkout
- **Interactive product cards** – Click to view details in a modal, or use the “BUY” button for instant checkout
- **Persistent shopping cart** – Items saved in `localStorage`, with live count badge
- **Smooth animations** – Fade‑in, slide‑up, and hover effects
- **Mobile‑first responsive design** – Looks great on any device
- **No external dependencies** – Pure HTML, CSS, and vanilla JS
